# **App Name**: Firebase Greetings

## Core Features:

- Display Greeting: Prominently display the fixed text 'hello firebase studio' on the screen.

## Style Guidelines:

- Primary color: A sophisticated, muted blue that suggests stability and professionalism. RGB hex: #2962A3 (derived from HSL: 220, 60%, 40%).
- Background color: A very light, almost white tone with a subtle hint of blue, creating a clean and open canvas. RGB hex: #ECF0F3 (derived from HSL: 220, 15%, 95%).
- Accent color: A brighter, dynamic cyan to add a touch of freshness and highlight potential interactive elements, contrasting effectively with the primary blue. RGB hex: #4BBFDB (derived from HSL: 190, 70%, 55%).
- Body and headline font: 'Inter' (sans-serif), chosen for its modern, clean, and highly legible appearance, fitting for a digital platform.
- The greeting text will be centrally aligned on the screen, providing immediate focus and simplicity.
